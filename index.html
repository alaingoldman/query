Intro

What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

JavaScript itself is understood by all browsers and doesn't require a download, but if you want to start using jQuery you have to send the library to the browser. You could either download and include jQuery as a resource or point to the jQuery library online by Google.

You can download jQuery here


      <head>
        <script src="jquery-1.11.2.min.js"></script>
      </head>

So you would include your jQuery library with the script tag. I want you to notice two things. The jQuery library file is a .js meaning it is written in javascript. Secondly, I want you to notice that the script tag is nested inside of the head tag.

If you don't feel like downloading jQuery alternatively Google hosts a jQuery file so you could point to to make your life easier. 


<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"> </script>

So... where does it go?

When you write jQuery or JavaScript you can add it to your document in one of two ways. Firstly, you can include it inside of a <script> tag.


    <script>
      $(document).ready(function(){
        $('.big a').hide()
      });
    </sript>

No need to worry about what this code does for now. Just know that A script tag can go wherever you want it to be in the HTML file however. It would make most sense to either be put at the top of the page in the <head> or the bottom of the page after all the HTML elements.

Secondly, you can also link to an external jQuery or JavaScript file with a script tag as well.


    <script src="example.js"> </script>

Here we are adding a file called example.js to our HTML document.


However, if you are intending to write jQuery make sure to add your jQuery code after the jQuery library or it will not work.

Syntax


In jQuery you select HTML elements with queries and perform actions on them. The basic syntax is:


   $(selector).action()
  • Dollar Sign ($) : This tells the browser access jQuery.
  • Selector : This queries (or finds) HTML elements.
  • Action : This is where you put the jQuery action that you want performed on the element(s).

So jQuery uses the same selector methodolgy that CSS uses to target HTML elements. So in order to understand jQuery lets review CSS selectors.


CSS Targeting with Selectors



        selector {
            property:value;
        }

In CSS you use a selector to target the elements you want to apply styles to. One way you can select and element is by looking for it's id or class attribute. If you want to select an element with the id value of content and another with the class value of wrap you would write this : 


                                
        #content {
          font-family: 'Lato', arial, sans-serif;
          font-size: 16px;
        }
        .wrap {
          margin: 0 auto;
        }

Here is a list of some other ways to target elements.


Select all elements on the page with * 

        *{margin:0;padding:0;}
Select based on the type of tag. 

        a {text-decoration:none;}
        body {background-color:red;}
Select descendents. 

        li a{
          text-decoration:none;
        }
You can join multiple selectors with a comma.
(Yes, that is a comma between .top and .bottom.) 

        .top ,.bottom{
          margin:10px;
        }
You can select all elements with a particular class by adding the element tag with the class and no space in between. 

        a.logo{
          float:left;

Selectors with jQuery


    $("p").hide()     // targets and hides all <p> elements.
    $(".test").hide() // targets and hides all elements with class="test".
    $("#example").hide() // targets and hides the element with id="example".

You don't have to worry what .hide(   ) is doing right now. Even though you can probably guess that it is hiding the element. But I want you to focus on the left side, being the selector.

So inside of single or double quotes we can use our selectors to target elements. In this basic example we targeted a p tag just with the letter p, much like how we would in CSS. We could also target based on id or class. With Id represented with a # in front of it and class represented with a . in front of it just like in CSS targeting.

Also as a side note if you noticed the way you make single line comment in javascript is
two forward slashes // This remains true for the jQuery library as well.


    $("p.large").hide()   // targets a <p> tag with the class="large".
    $(".sml, .xl").hide() // targets an element with class="sml"
                          // and also targets an element with class="xl".

    $(".big a").hide() // targets an <a> tag within an element with class="big".
    $("li:nth-of-type(3)").hide() // targets the 3rd <li>

jQuery selectors can be very broad or specifc depending on the given situtation.

Using Functions 

    function name(parameter1, parameter2, parameter3) {
        // code to be executed
    };

If there is a group of code you want to perform mulitple times instead of writing it over and over we would use a function. This allows us to call a group of code with one command.

Here name is the name of the function. It can be called anything you want and this becomes a keyword to call the function. Later on we can call with function by just typing name().

Parameters, also know as arguments, are an optional field you can include when defining a function. If your function for example adds two numbers together it would require 2 parameters being the first number and the second number. These paramenters will be available inside of the function.

The code to be executed goes inside of the two curly braces.

Now in order to start the function you would call it by typing it's name with the parameters.
name(   p1,   p2,   p3   ) If the parameter section is blank you leave the parathesis blank name( )


    function daysAlive(age) {
        var result = age * 365;
        alert("You have been alive " + age + " days!");
    };

    var answer = prompt("How old are you?");
    daysAlive(answer);

So I created a function called daysAlive which takes one parameter of age. It then performs a calculation of multiplying that age by 365 and alerting that back to the user. Now that was the definition of the function. We have to actually call it. So we do a prompt to get the user's age and then we throw that into the function. What I want you to notice is that the variable answer is thrown into the function and the function parameter is called age. It doesn't matter what the name of the variable is as long as daysAlive has a value in the parameter section. I could have instead done this daysAlive( 27 ); with the same effect.

    $("#cheese").click(function(){

    });

Now inside of .click( ) we create a function which alerts that this div has been clicked.

Making a Slider

Ever see those fancy sliders where images slowly slide by. Well thats jQuery. So lets test what we've learned and try to make one.


So how do we do this?


So before we can start the slider animation we have to understand what a slider is conceptually then in terms of HTML + CSS.

So a slider is a series of images that slowly pan in from the right pushing the other out of the way.


Problem 1 : HTML elements naturally stack on top of each other. They don't innately stack side by side.

Problem 2 : The images to the left of the current image cannot be seen until they pan in.

Problem 3 : How do we actually do the animate?

Problem 4 : If all the images pan to the left how do we put the left most image back so we don't run out of images.


Stack the images


Now if i throw two images here obviously they will stack ontop of each other so lets see if we can get several images side by side with CSS.



  <div id="slider-wrap">
    <img src="bb1.jpg">
    <img src="bb2.jpg">
    <img src="bb3.jpg">
    <img src="bb4.jpg">
  </div>




The images go down as expected. Lets see if we can change that.


  <div id="slider-wrap">
    <img src="bb1.jpg" id="first">
    <img src="bb2.jpg" id="second">
    <img src="bb3.jpg" id="third">
    <img src="bb4.jpg" id="fourth">
  </div>
                    

    .slider-wrap{
      min-height: 500px;
      position: relative;
    }
    .slider-wrap img{
      width:100%;
      position: absolute;
    }
    #first{}
    #second{left:100%;}
    #third{left:200%;}
    #fourth{left:300%;}



Ok cool so they are next to each other. Now we just need to hide the ones that overlap the side. This is very easy. All you have to do is add overflow:hidden; So the finished CSS should look like this.


    .slider-wrap{
      min-height: 500px;
      position: relative;
      overflow:hidden; /* JUST ADDED */
    }
    .slider-wrap img{
      width:100%;
      position: absolute;
    }
    #first{}
    #second{left:100%;}
    #third{left:200%;}
    #fourth{left:300%;}



Done :)


Ok... how do we animate this?


Lets just scootch over #first, #second, #third and #fourth.


    $('#first')
        .animate({left: '-=100%'}, 3000)
    $('#second')
        .animate({left: '-=100%'}, 3000)
        .animate({left: '-=100%'}, 3000)
    $('#third')
        .animate({left: '-=100%'}, 3000)
        .animate({left: '-=100%'}, 3000)
        .animate({left: '-=100%'}, 3000)
    $('#fourth')
        .animate({left: '-=100%'}, 3000)
        .animate({left: '-=100%'}, 3000)
        .animate({left: '-=100%'}, 3000)

This makes the animation happen too fast lets add a delay to it. Lets pause each animation for 1 second. We can use delay( ); 


    $('#first')
        .animate({left: '-=100%'}, 3000)
    $('#second')
        .animate({left: '-=100%'}, 3000).delay(1000)
        .animate({left: '-=100%'}, 3000)
    $('#third')
        .animate({left: '-=100%'}, 3000).delay(1000)
        .animate({left: '-=100%'}, 3000).delay(1000)
        .animate({left: '-=100%'}, 3000)
    $('#fourth')
        .animate({left: '-=100%'}, 3000).delay(1000)
        .animate({left: '-=100%'}, 3000).delay(1000)
        .animate({left: '-=100%'}, 3000)
